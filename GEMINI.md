# StandardPythia Art Advisory

Minimalist, static website for Francesca De Filippo's art advisory practice, StandardPythia.

## Project Overview

- **Purpose:** Professional portfolio and contact page for art advisory services.
- **Technologies:** HTML5, Vanilla CSS3 (Custom Properties, CSS Grid, Flexbox).
- **Architecture:** Single-page static site (`pythia.html`).

## Directory Structure

- `index.html`: Main entry point containing content, styles, and structure.
- `GEMINI.md`: Instructional context for Gemini CLI.

## Development

### Running
Open `index.html` in any modern web browser.

### Styling
- **Theme:** Dark emerald green (`#0e3931`) with high-contrast white/grey typography.
- **Font:** Helvetica-based stack.
- **Responsiveness:** Mobile-first considerations via Media Queries (< 900px).
- **Placeholders:** Uses CSS-styled divs for editorial image placeholders (4:5 and 16:9 ratios).

### Conventions
- Use CSS Variables (`:root`) for global styling.
- Maintain the "curatorial rigor" aesthetic: minimalist, plenty of whitespace, and discreet typography.
- Avoid external dependencies (keep it Vanilla).
